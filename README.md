## Setup dependencies ##

	make

## Build ##

	make build

## Develop ##

Make sure you have [Leiningen](https://github.com/technomancy/leiningen/#installation) installed.

Console 1:

	rlwrap lein figwheel

Console 2:

	./build/node/bin/node target/server_dev/bitwalden_node.js
